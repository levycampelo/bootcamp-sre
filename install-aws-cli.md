## Atualizar pacote
```bash
$ sudo apt-get update
```

## Instalar dependencias
```bash
$ sudo apt-get install unzip curl -y
```

## Baixar awscli
```bash
$ curl "https://awscli.amazonaws.com/awscli-exe-linux-x86_64.zip" -o "awscliv2.zip"
```

## Descompactar
```bash
$ unzip awscliv2.zip
```

## Instalar
```bash
$ sudo ./aws/install
```

## Verificar versao
```bash
$ aws --version
```
