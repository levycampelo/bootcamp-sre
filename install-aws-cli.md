## Atualizar pacote
```sudo apt-get update```

## Instalar dependencias
```sudo apt-get install unzip curl -y```

## Baixar awscli
```curl "https://awscli.amazonaws.com/awscli-exe-linux-x86_64.zip" -o "awscliv2.zip"```

## Descompactar
```unzip awscliv2.zip```

## Instalar
```sudo ./aws/install```

## Verificar versao
```aws --version```
