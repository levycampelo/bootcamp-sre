## Atualizar pacote
```bash
$ sudo apt-get update
$ sudo apt-get install unzip curl -y
```

## baixar/instalar awscli
```bash
$ curl "https://awscli.amazonaws.com/awscli-exe-linux-x86_64.zip" -o "awscliv2.zip"
$ unzip awscliv2.zip
$ sudo ./aws/install
```

## Verificar versao
```bash
$ aws --version
```
